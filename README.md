# 🛰️ CanSat Ground Control System

A web-based Ground Control System developed for real-time CanSat mission monitoring and simulation.

## Features

- 📡 Live telemetry monitoring
- 📈 Real-time graphs (Chart.js)
- 🗺️ GPS tracking (Leaflet.js + OpenStreetMap)
- 🧭 3D orientation visualization (Three.js)
- 📷 Camera integration
- 🚨 Mission error monitoring
- 🎮 Mission command panel
- 📄 CSV & graph export
- 💻 Mission console logging

## Technologies

HTML • CSS • JavaScript • Chart.js • Leaflet.js • Three.js

## Run

Open `index.html` in a web browser (or use Live Server in VS Code).
